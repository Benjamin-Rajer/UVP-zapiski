Delali bomo še naprej analizo podatkov.

Ko računalniki se povezujejo s strežniki preko interneta, za podatke pošljejo nekaj, čemur se reče get request. To je kot neka prošnja za dostop do podatkov. Ta get request gre preko posebnega strežnika, ki se imenuje DNS server. Ta server je kot neke vrste telefonski imenik za celoten internet in vsebuje domene spletnih strani. Prenos podatkov deluje po protokolu HTTP, za bolj varen prenos se dandanes zelo pogosto uporablja HTTPS protokol.

Obstaja še post request, ki strežniku še nekaj zraven pošlje, npr. ko se moramo logirat notri v neko spletno stran.

Obstajajo knjižnice, ki nam omogočijo, da znotraj Pythona delamo requests.

```python
>>> import requests
>>> r = requests.get("https://www.youtube.com")
>>> r.status_code # status kode povedo, če je vse ok z requestom
200 # 200 pomeni da je ok, npr. 404 ni okej
>>> r.text # to nam bo dal nek string, ki je html koda spletne strani
>>> r.headers # v glavi so osnovni podatki o našem requestu
```

Obstajajo strežniki in naslovi, ki niso namenjeni za nas, ampak so namenjene samo za pridobivanje informacij v obliki kode - pogosto je kategorizirano kot API - application programming interface.

Določene spletne strani se potrudijo, da ne pobiramo podatke dol s spletnih strani, ker bi lahko z milijoni get requesti močno obremenili spletno stran.
Nekateri dodajajo tudi dodaten mehanizem, da lahko samo preko brskalnikov delamo get requeste. Get request vrne še dodatno JavaScript kodo, ki se izvede samo v brskalnikih in ne v samostojnih programih. Vendar to se da tudi zaobiti.


Probajmo najprej na eno malo manj zavarovano spletno stran: goodreads

Želimo najti vse podstrani, ki vsebujejo podatke o knjigah:

```python
import requests
import time # da malo počakamo med get requesti

HEADERS = {"User-Agent": "Mozilla/5.8"}
odgovor = requests.get("https://www.goodreads.com/list/show/1.Best_Books_Ever", headers=HEADERS)
print(odgovor)
# morali smo dodati še te headers, da zaobidemo zahteve strežnika
# dajmo še pobrati nekaj podstrani z uporabo for zanke
ST_STRANI = 5
for i in range(1, ST_STRANI):
	odgovor = requests.get(f"https://www.goodreads.com/list/show/1.Best_Books_Ever?page={i}", headers=HEADERS)
	if odgovor.status_code != 200:
		print("napaka", i)
	else:
		vsebina = odgovor.text
		with open(f"datoteka{i}.txt", "w") as dat:
			dat.write(vsebina) # zdaj smo dali zadeve iz spletnih strani v svoje datoteke
		time.sleep(1) # lahko malo tudi počakamo
			
```

Selenium nam pomaga zaobiti omejitve strežnika, da naredi get request in se pretvarja, da je brskalnik.

```python
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys
import time

driver = webdriver.Chrome()

driver.get("https....") # naloži spletno stran

time.sleep(10) # moramo malo počakati vmes, ker upravljamo z brskalnikom in da lahko tudi karkoli vidimo

gumb = driver.find_element(By.CLASS_NAME, "ipc-see-more__button") # iskanje po html kodi
gumb.send_keys(Keys.RETURN) # lahko tudi pritisnemo gumb na spletni strani
# lahko bi tudi naredili for zanko in ponavljali da pritiska ta gumb
time.sleep(5)

print(driver.page_source)
```
