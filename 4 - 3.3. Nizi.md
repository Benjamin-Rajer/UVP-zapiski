Nizom se v angleščini reče strings.

<span style="color:rgb(0, 176, 80)">Nizi</span> so v Pythonu način, kako shranjevat podatke. Nize pišemo v narekovajih.
Obstaja funkcija <span style="color:rgb(0, 176, 240)">len</span> (length), ki vrne dolžino niza.
Nize lahko tudi seštevamo, to jih doda skupaj, s tem ustvarimo nov niz, ki je sestavljen iz obeh.
Množenje nizov ni definirano, lahko pa zmnožiš niz z nekim naravnim številom, da ga večkrat sešteješ skupaj.

Pogoj <span style="color:rgb(0, 176, 240)">in</span> pogleda, ali je neko zaporedje črk v nekem nizu.
```python
>>>niz = "abcd"
>>>"abc" in niz
True # vrne True, ker abc je v nizu
```

Definirajmo funkcijo, ki pogleda, koliko samoglasnikov je v nizu(probamo brez for zanke - z rekurzijo).

```python
def stevilo_samoglasnikov(niz):
	if len(niz) == 0:
		return 0 # ustavitveni pogoj
	
	stevilo = 0
	if niz[0] in "aeiouAEIOU":
		stevilo += 1 # tako povečamo neko vrednost za 1: znak +=
	stevilo += stevilo_samoglasnikov(niz[1:])
	return stevilo
```

Še ena funkcija: narisali bomo graf implicitne funkcije:
Imamo npr. funkcijo $f(x,y) = x^2 + y^2 - 1$ in narišemo funkcijo na grafu y(x). Razdelimo graf na kvadratke in v tistih, kjer je funkcija dovolj blizu ničle, bomo narisali zvezdico.

```python
def graf_implicitne_funkcije(f, x1, x2, y1, y2, n, m):
	#x1, x2, y2, y1 so meje grafa, kjer rišemo , n je št kvadratkov v vrstici, m je št. vrstic
	# razmislimo, kako bi funkcija sebe poklicala na manjšem problemu - lahko ko sprintamo nekaj vrstic, zmanjšamo y interval
	if m == 0:
		return 
		
	narisi_vrstico(f, x1, x2, y2, n)
	graf_implicitne_funkcije(f, x1, x2, y1, y2 - ((y2-y1) / m), n, m-1) #y2 manjši za za toliko kolikor je razdeljena razlika med y2 in y1 na m vrstic - eno vrstico kvadratkov odstranimo

def narisi_vrstico(f, x1, x2, y2, n): # zdaj smo v eni konkretni vrstici
	if n == 0:
		print("") # za narediti novo vrstico, spodaj pazimo da ne gre v novo
		return
		
	if abs(f(x1,y2)) < 0.1: # to lahko tudi spremenmo, to je nek parameter
		print("*", end="") # da ne gre v novo vrstico
	else:
		print(" ", end="")
		
	narisi_vrstico(f, x1 + ((x2-x1) / n), x2, y2, n-1) # spet z rekurzijo zmanjšamo problem, povečamo x1
	
def kroznica(x,y):
	return x**2 + y**2 - 1 # ko bo to blizu 0, bo funkcija na krožnici

def polinom(x,y):
	return y - x * (x-1) * (x+1)
```

Naslednje bomo narisali fraktal - Mandelbrotova množica.

Mandelbrotovo množico imamo v kompleksni ravnini. Notri so tista kompleksna števila $c$, za katera funkcija $f_{c} (z)  = z^2 + c$ ko vstavimo notri 0 in damo večkrat čez funkcijo, torej $f_{c}(0), f_{c}(f_{c}(0)), \dots$ ne zbeži v neskončnost.

Narišimo preprosto Mandelbrotovo množico v Pythonu:

```python
def mandelbrot_f(z1, z2, c1, c2): # realni in imaginarni deli z-ja in c-ja
	return (z1**2 - z2**2 + c1, 2 * z1 * z2 + c2) # tole vrne funkcija - z^2+c
	
	
def ponovi(z1, z2 , c1, c2, n):
	if n == 0: # ko bomo ponovili n-krat, bomo vrnili te vrednosti
		return z1, z2
		
	noviz1, noviz2 = mandelbrot_f(z1, z2, c1, c2) # damo skozi funkcijo z^2+c
	
	if noviz1**2 + noviz2**2 > 500: # da ne bo preveliko bomo prej ustavili
		return noviz1, noviz2
		
	return ponovi(noviz1, noviz2, c1, c2 ,n-1) # rekurzivno ponavljamo postopek da dajamo notri v funkcijo


def mandelbrot(x,y):
	z1, z2 = ponovi(0, 0, x, y, 100) # to damo v funkcijo - začnemo s številom 0 in ponavljamo 100-krat
	if z1**2 + z2**2 > 500: # če je kvadrat dolžine ogromen, potem vrnemo 1 in se zvezdica ne bo narisala
		return 1
	else:
		return 0
```
