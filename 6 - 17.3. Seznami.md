Seznami (angl. lists) so način v Pythonu, ki nam omogočajo, da si shranjujemo stvari. 
Seznam naredimo z oglatimi oklepaji, člene ločimo z vejico.
Python za seznam rezervira en kos spomina za nas in notri shrani te naše elemente.

```python

```

V sezname lahko damo karkoli od podatkovnih tipov, lahko tudi mešane stvari.

```python

```

Dajmo narediti funkcijo, ki najde največji element v seznamu (privzamemo da ima seznam samo števila):

```python
def najvecji(sez):
	max = None # damo None, bolje kot 0 - če imamo samo negativna števila, 0 ni ok
	
	for j in sez:
		if max == None or j > max: # ne moremo preverjati None s številom
			max = j
			
	return max
```

Še en primer: konstantna matrika (matrika katere členi so vsi enaki)

```python
def konstantna_matrika(n, m, x): # n-št.vrstic, m-št.stolpcev, x-člen matrike
	return n * [m * [x]] # najprej v eni vrstici je m elementov, potem pa n vrstic

	
```

Problem - če zamenjamo prvi element v matriki, bo se cel stolpec spremenil - ker smo pomnožili vrstico z n in smo n - krat samo ponovili mesto v spominu.

Obstaja boljši način, da postanejo elementi neodvisni:

```python
def konstantna_matrika(n, m, x):
	mat = [] # najprej naredimo prazno matriko
	for _ in range(n): # naredimo for zanko
		mat.append(m * [x]) # vedno naredi novo ločeno vrstico
	
	return mat
```

Dajmo urediti seznam po velikosti. To bomo naredili s pomočjo enega algoritma - našli bomo največjega in ga dali na konec seznama itd. Probajmo z rekurzijo:

```python
def uredi(sez):
	# vrne nov seznam z elementi, urejenimi po velikosti
	
	if sez = []: # ustavitveni pogoj
		return []
	nov_sez = sez.copy()
	i = indeks_najvecjega(sez) # glej nazaj funkcijo
	
	del nov_sez[i] # izbrišemo ta najmanjšega
	
	urejen = uredi(nov_sez) # tole je rekurzija - pogledal bo zdaj drugi največji element itd. - najprej bo appendan ta najmanjši, potem ta drugi najmanjši itd.
	urejen.append(sez[i])
	
	return urejen
```

To je malo zamudno, probajmo kak drugi sortirni algoritem:

Bubble sort je algoritem, pri katerem vsak pogleda svojega soseda in če je sosed z večjim indeksom manjši, se zamenjata.

```python
def bubble_sort(sez):
	# pa dajmo tukaj kar seznam sam urediti ne da bi novega naredili
	while True:
		so_urejeni = True # dodamo še boolean za ustavitveni pogoj - ko bodo urejeni bo to ostalo True
		
		for i in range(len(sez) - 1): # gremo spet po indeksih brez zadnjega
			if sez[i] > sez[i+1]: # če je element večji od naslednjega, zamenjamo
				sez[i], sez[i+1] = sez[i+1], sez[i]
				so_urejeni = False # niso še bili urejeni, tako da damo zdaj false
				
		if so_urejeni:
			break # break se nanaša na while zanko
	
	# ta funkcija nič ne returna, ker samo uredi seznam, seznam bo tudi zunaj funkcije potem urejen
```

Še en algoritem:

Quicksort je algoritem, ki prvo seznam razbije na dva (idealno enako velika) dela in vsak kos naprej rekurzivno razbije na dva dela. Kolikokrat moramo narediti je približno velikosti log(n).

Vzamemo naključen element, primerjamo elemente in jih razvrstimo v dva dela s primerjavo s tem naključnim elementom.

```python
import random

def quicksort(sez):
	if len(sez) <= 1: # ustavitveni pogoj za rekurzijo
		return sez
	index = random.randint(0, len(sez)-1) # vzamemo naključni indeks za ta element primerjati
	pivot = sez[index] # temu primerjalnemu elementu se reče pivot
	
	manjsi = []
	vecji = [] # v ta dva seznama bomo razbili
	for j in range(len(sez)): # sprehodimo se po indeksih seznama
		if j == index:
			continue # ne primerjamo ta istega
		el = sez[j]:
		if el >= pivot:
			vecji.append(el)
		else:
			manjsi.append(el)
	
	return quicksort(manjsi) + [pivot] + quicksort(vecji) # razbili smo na dva dela in rekurzivno ponavljamo za ta dva dela - sestavil se bo skupaj urejen seznam
```

Quicksort je hitrejši od prejšnjih dveh algoritmov.
