<span style="color:rgb(0, 176, 80)">Rekurzija</span> je način kako funkcija s klicanjem same sebe sestavi postopek za reševanje nekega problema. Ko kliče samo sebe, se pogoji funkcije spremenijo, dokler ne pride do nekega baznega primera, kjer se ustavi.

Najbolj znan primer je računanje fakultete števila:

```python
def fakulteta(n):
	if n <= 1:
		return 1
	else:
		return n * fakulteta(n-1)		
```

Kot vidimo, smo uporabili pri računanju rekurzijo, torej smo s tem fakulteto rekurzivno definirali. Kar bo funkcija fakulteta naredila je v resnici to:
Ko ji damo število n bo postopoma sestavljala tak rezultat:

```python
n * fakulteta(n-1)
n * n-1 * fakulteta(n-2)
n * n-1 * n-2 * fakulteta(n-3)
# na koncu dobimo:
n * n-1 * n-2 * ... * fakulteta(1) # zdaj pa pride bazni primer
n * n-1 * n-2 * ... * 1 
```

Torej funkcija fakulteta kliče še enkrat funkcijo fakulteta, samo da se n spremeni v n-1 in vedno ko se izvede, se pokliče še enkrat, dokler ne pridemo do baznega primera za n = 1.

Še en primer: Fibonaccijeva števila:

```python
def fibonacci(n):
	if n == 0:
		return 0
	elif n == 1:
		return 1
	else:
		return fibonacci(n-1) + fibonacci(n-2)
```

Vsako Fibonaccijevo število je vsota svojih dveh predhodnikov. Z rekurzijo lahko sestavimo vsa Fibonaccijeva števila od 0. mesta naprej. Vendar računati števila tako lahko hitro postane zamudno, ker v vsakem koraku kličemo funkcijo 2x.
Zato lahko definiramo Fibonaccijevo zaporedje še na en drug način(s poljubnima začetnima členoma a in b):
$$a,\; b,\; a+b,\; b+(a+b) = a+2b, \; (a+b) + (a+2b) = 2a + 3b,\; \dots$$
```python
def splosni_fibonacci(n, a=0, b=1): # vrne n-ti člen zgornjega zaporedja
	if n == 0:
		return a
	elif n == 1:
		return b
	else:
		return splosni_fibonacci(n-1, b, a+b) # naslednja dva člena zaporedja sestavimo iz a in b - kot da bi za eno zamaknili naprej zaporedje
```

Kaj smo naredili pri splosni_fibonacci(n-1, b, a+b)? Program prestavi to zaporedje naprej za eno mesto s tem, da vzame ta tretji argument, ga prestavi v drugega, drugega in tretjega pa sešteje in ga da v tretjega. Prvi argument šteje samo, koliko korakov smo naredili.

Torej to bi se nadaljevalo po istem postopku in bi dobili potem za argumente (n-2, a+b, b+(a+b)), isto kot se dogaja pri samem zaporedju, torej da seštejemo prejšnja dva člena. In tako potem se premikamo naprej po zaporedju gor, dokler ne pridemo do n = 1, kadar dobimo naš rezultat, ki je zapisan kot tretji argument.

Opomba: 
1. Ko kličemo funkcijo, moramo kot argument podati samo število n, ker sta ostali dve že privzeti kot 0 in 1, lahko pa se odločimo ju spremeniti in dati kaj drugega kot začetni vrednosti:
```python
splosni_fibonacci(10) # to nam bo dalo 10. Fibonaccijevo število
splosni_fibonacci(10, 2, 5) # to nam da 10. število v zaporedju z začetnima številoma 2 in 5 namesto privzetih 0 in 1
```

<span style="color:rgb(0, 176, 80)">Algoritem</span> je zaporedje korakov, s katerimi dobimo iskani rezultat. Je postopek, s katerim izračunamo želene vrednosti.

Eden najstarejših algoritmov na svetu je Evklidov algoritem, ki najde GCD (največji skupni delitelj) dveh števil. Kako in zakaj deluje postopek? Poglejmo:
Naj bosta $m, n$ števili, za kateri iščemo največji skupni delitelj in $d$ ta največji skupni delitelj. 
Pišimo: $m = k \cdot n + o$, kjer je $0\leq o < n$. $d$ deli $k\cdot n$ in $m$, torej deli tudi $o = m - k \cdot n$. Potem isto nadaljujemo, samo namesto $m$ in $n$ vzamemo $n$ in $o$ (GCD se potem ohranja, ker vsi skupni delitelji $m$ in $n$ so tudi delitelji $o$). To ponavljamo, dokler ne pridemo do tega, da $o$ postane enak 0. Takrat je največji skupni delitelj $n$ in $o$ kar enak $n$.

Če to napišemo v Pythonu:

```python
def gcd(m,n):
	if n < m:
		m, n = n, m # zamenjamo, da je m večji ali enak n, niti ni potrebno
	if n == 0:
		return m
	else:
		return gcd(n, m % n) 
```

Še krajše se da napisati:

```python
def gcd(m,n):
	return m if n == 0 else gcd(n, m % n)
```

def integriraj(f,a,b,n): # n je število stolpcev
	if n == 0:
		return 0
	sirina_stolpca = (b-a) / n
	stolpec = sirina_stolpca * f(a + sirina_stolpca / 2) # vzamemo vrednost funkcije na sredini stolpca
